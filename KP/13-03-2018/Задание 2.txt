Задание 2 - создание сервера.
1. Создаем и инициализируем проект:
$ mkdir node_server
$ cd node_server
$ npm init -D

2. Устанавливаем зависимости:
npm i -D nodemon (nodemon - автоматически перезапускает проект после сохранения изменений в коде).
npm i moment (moment - модуль работы со временем)

3. В файле package.json в разделе scripts вносим изменение: “start”:”nodemon”
4. Создаем файл index.js
5. Запускаем сервер: node index.js 

См. приложение 2 и 3
